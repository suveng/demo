# 开源脚手架

## 地址

[Github](https://github.com/suveng/demo/tree/master/chapter.001)

## v1.0功能

- springboot2.1.2
- druid连接池配置
- MySQL
- mybatis
- fastjson
- 统一IDE格式化代码
- 统一日志打印
- 全局请求id
- 全局异常处理
- 业务告警: 钉钉群消息告警
